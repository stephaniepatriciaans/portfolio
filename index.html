Projects

Home Price Predictions → Real Estate ML Pipeline

Predicting U.S. home prices using 80k+ housing records from CRMLS & Trestle API.

  • Built a full machine learning pipeline for residential price prediction.
  • Performed EDA, data cleaning, and feature engineering (log transforms, interaction terms).
  • Tuned Ridge Regression & Random Forest models achieving R² = 0.89 and MDAPE ≈ 6%.
  • Outcome: Identified top pricing drivers and improved valuation transparency.

Tech Stack: Python, Pandas, Scikit-learn, Matplotlib, NumPy, REST APIs (Trestle/Cotality), Tableau

TrueFork → Recipe Ratings & Nutrition Bias Analysis

Analyzing whether healthy recipes are rated differently using 1M+ Food.com reviews.

  • Designed an end-to-end ML pipeline from data ingestion to interpretation.
  • Engineered features from text sentiment (NLTK/VADER), ingredient complexity, and nutrition facts.
  • Built models (Linear Regression, Random Forest, Ridge/Lasso) with SHAP explainability.
  • Outcome: Evidence of bias against low-calorie/low-sugar foods.

Tech Stack: Python, Pandas, Scikit-learn, SHAP, Matplotlib, Seaborn, Jupyter

Project Loan Charge-Offs → Credit Risk Analysis

Predicting loan charge-offs to improve lending decisions.

  • Engineered DTI ratios, credit score groups, and loan terms.
  • Trained Logistic Regression, Decision Trees, and Gradient Boosting.
  • Outcome: AUC > 0.80, highlighting top default predictors.

Tech Stack: Python, Pandas, Scikit-learn, XGBoost, Matplotlib
